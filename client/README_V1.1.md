# Voice Input Framework v1.1 使用说明

## 🎉 新功能介绍

### 1. 精细化快捷键设置

**支持区分左右修饰键**：
- ✅ Left Shift / Right Shift
- ✅ Left Alt / Right Alt (Option on macOS)
- ✅ Left Ctrl / Right Ctrl
- ✅ Left Cmd / Right Cmd (macOS)

**快捷键录制功能**：
1. 点击"录制"按钮
2. 按下目标快捷键
3. 自动识别并填充

**预设方案**：
- 默认：Right Alt + V（推荐）
- 游戏模式：F13
- 左手模式：Left Alt + V
- 经典模式：Alt + V（不区分左右）

### 2. 系统托盘支持

**功能**：
- 最小化到系统托盘
- 托盘菜单快速操作
- 托盘图标状态指示

**使用方法**：
1. 点击"最小化到托盘"按钮
2. 右键点击托盘图标查看菜单
3. 通过菜单控制录音和切换模型

**状态图标**：
- 🔘 灰色：就绪
- 🔴 红色：录音中
- 🔵 蓝色：处理中
- ⚠️ 感叹号：错误

### 3. 悬浮录音指示器

**功能**：
- 录音时显示小型悬浮窗口
- 半透明效果
- 可拖动定位
- 实时显示录音时长

**使用方法**：
1. 勾选"使用悬浮录音指示器"
2. 开始录音时自动显示
3. 可以拖动到任意位置
4. 停止录音后自动消失

## 🚀 快速开始

### 安装依赖
```bash
pip install pystray Pillow
```

### 启动客户端
```bash
cd ~/voice-input-framework
python client/gui_v2.py
```

## 📖 详细配置

### 快捷键设置

**方法一：手动输入**
- 在快捷键输入框中输入，如：`right_alt+v`
- 点击"更新"按钮

**方法二：录制**
- 点击"录制"按钮
- 按下目标快捷键
- 自动填充

**方法三：预设方案**
- 从下拉菜单选择预设
- 点击"应用预设"

### 托盘设置

**启动时最小化**：
- 勾选"启动时最小化到托盘"
- 程序启动后自动隐藏到托盘

**托盘菜单**：
- 显示窗口：恢复主窗口
- 隐藏窗口：最小化到托盘
- 开始/停止录音：控制录音
- 模型：切换识别模型
- 退出：关闭程序

### 指示器设置

**悬浮录音指示器**：
- 勾选"使用悬浮录音指示器"
- 录音时显示小型窗口
- 显示录音状态和时长

## 🔧 故障排除

### 托盘图标不显示
**原因**：缺少 `pystray` 或 `Pillow`

**解决**：
```bash
pip install pystray Pillow
```

### 快捷键不生效
**原因**：
1. 快捷键格式错误
2. 快捷键被其他程序占用

**解决**：
1. 使用录制功能自动识别
2. 尝试其他快捷键组合

### 悬浮指示器不显示
**原因**：PySimpleGUI 版本过低

**解决**：
```bash
pip install --upgrade PySimpleGUI
```

## 💡 使用技巧

1. **推荐快捷键**：Right Alt + V
   - 不影响左侧 Alt 的系统功能
   - 方便右手操作鼠标的用户

2. **游戏模式**：F13 单键
   - 不占用常用组合键
   - 适合游戏时使用

3. **托盘模式**：
   - 适合长时间后台运行
   - 通过托盘快速控制

4. **悬浮指示器**：
   - 录音时无需切换窗口
   - 实时查看录音状态
